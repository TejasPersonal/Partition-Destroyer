# Partition Destroyer

⚠️ Tool to delete partitions and create partition tables for storage devices / drives with ease.

## Requirements

- Linux
- Root privileges
- Python 3

## Setup
```bash
git clone https://github.com/TejasPersonal/Partition-Destroyer.git
cd Partition-Destroyer

python -m venv env
source env/bin/activate
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
